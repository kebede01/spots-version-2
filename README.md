# The project name
This project is named 'SPOTS-VERSION-2'.

## Project description and functionality
The project displays photos posted with __*various  screen sizes*__ taken into consideration.
Its function is to __add__ photos to this app which functions as ***photo album***.

## A description of the technologies and techniques used
The technologies used in this project are HTML, CSS. And the techniques used to make it responsive to different screen sizes are disply grid and display flex css properties. in particular grid ***auto-fit*** property was instrumental in dynamically adjusting column numbers.
```
.cards__list {
  display: grid;
  grid-template-columns: repeat(auto-fit, 413px);
  }
  ```
